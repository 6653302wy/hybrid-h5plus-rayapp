微信网页授权API

  • 网页授权用户信息(包含nickName等)
    • 地址:http://app.epoint.com.cn/Epoint_Weichat_Platform/redirectToEpointWeichatOauthSysInfo.do?targetUrl=http://app.epoint.com.cn/showcase.dcloud.rayapp/html/weichat/demo_weichat_oauth_simple.html
    • 注意:这里identify没填,默认为公司服务号,地址为该测试页的公网地址
  • 网页授权openId(只有openId)
    • 地址:http://app.epoint.com.cn/Epoint_Weichat_Platform/redirectToEpointWeichatOauthOpenId.do?targetUrl=http://app.epoint.com.cn/showcase.dcloud.rayapp/html/weichat/demo_weichat_oauth_simple.html
  • 企业号授权用户信息
    • 地址:http://app.epoint.com.cn/Epoint_Weichat_Platform/redirectToEpointWeichatOauthEnterprise.do?targetUrl=http://app.epoint.com.cn/showcase.dcloud.rayapp/html/weichat/demo_weichat_oauth_simple.html
    • 注意:这里identify没填,默认为公司企业号,地址为该测试页的公网地址
  • 网页授权换取code(只有code和identify)
    • 地址:http://app.epoint.com.cn/Epoint_Weichat_Platform/redirectToEpointWeichatOauthCode.do?targetUrl=http://app.epoint.com.cn/showcase.dcloud.rayapp/html/weichat/demo_weichat_oauth_simple.html
  • 通过code获取用户信息接口(自行加上微信回调)
    • 地址:https://open.weixin.qq.com/connect/oauth2/authorize?appid=wxc2b37ed7397251c7&redirect_uri=http%3A%2F%2Fapp.epoint.com.cn%2FEpoint_Weichat_Platform%2FcheckEpointWeichatOauthSysInfo.do%3FtargetUrl%3DTARGETURL&response_type=code&scope=snsapi_userinfo&state=STATE#wechat_redirect
    • 注意:这里微信回调的地址是自己加上去的,这里用的是公司服务号的授权
